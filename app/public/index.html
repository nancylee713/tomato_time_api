Welcome to Tomato Time API!

Introduction

This is the documentation for version 1 (V1) of the Tomato Time API, created by Joshua Sherwood, Mack Halliday, Scott Payton, and Nancy Lee

Trivias in this API were imported from the Open Trivia Database. The Tomato Time API is designed to be used in conjunction with the quiz app Tomato Time.

Response Formats

The default response format is JSON application/json.

Authentication

The API is open to public. Feel free to create fun games with it!

All Routes

All routes need to be prefixed with https://peaceful-chamber-62417.herokuapp.com/api/v1/questions

URL Query Parameters

Param Type Examples
category string Valid categories: General Knowledge, Entertainment: Books, Entertainment: Film, Entertainment: Television, Entertainment: Music, Science & Nature, Geography History
difficulty string Valid levels: easy, medium, hard
amount number Limit to 50 per request
